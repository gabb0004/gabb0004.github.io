About Bootstrap

What is Bootstrap?

Bootstrap is an open-source CSS framework intented to be used for mobile-first web developpment. It uses CSS and JavaScript-based templates to make creating different elements easier for the developper.

Who runs Bootstrap?

It was originally created by a developper at Twitter in 2010, but now Bootsrtap has an independant dev team on Gtihub who regularly maintains it.

Why should I use Bootstrap?

Bootsrtap enables the developper to make a responsive webpage as easy and frictionless as possoble. All of your code can be done in HTML with zero CSS or JavaScript, since the dev team at Bootsrtap has done it all for you.

A jumbotron is used to display important content on your website in a large fashion. It's generally used for banners or ads.
